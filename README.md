# Angular PWA

setup

```bash
ng new angular_pwa --routing true --style scss
```

add angular pwa

```bash
ng add @angular/pwa
# ng add @angular/pwa --project angular_pwa
```

```bash
ng build
```

```bash
ng build
http-server -p 8080 -c-1 dist/angular_pwa
```
